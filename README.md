## 自行编译

依赖项：
 * [node.js](https://nodejs.org/)
 * gitbook-cli
```shell
$ npm install gitbook-cli -g
```
 * gitbook-plugin-fancybox
```shell
$ npm install --save gitbook-plugin-fancybox
```

然后 clone 本书
```shell
$ git clone https://github.com/hz-ingress/ingress-tutorial.git
```

接下来你可以选择其中一种方法来查看本书的内容：

 * 构建静态 HTML
```shell
$ gitbook build
```

 * 直接在浏览器中观看
```shell
$ gitbook serve
```
