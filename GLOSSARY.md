# ADA Refactor
俗称蓝毒，用于把 Enlightened 控制的 Portal 转为 Resistance

# Agent
特勤，玩家的通称

# Deploy
俗称插，把 Resonator 放置在 Portal 上的操作

# Faction
阵营，玩家所属的游戏双方的总称

# Enlightened
启示军，俗称绿军，阵营之一

# Field
控制场，俗称 F，由三个同阵营 Resonator 两两 Link 形成的区域

# Force Amp
一种能增加对敌对阵营 Agent 攻击强度的 Mod

# Glyph Hack
连位符入侵，俗称画图，通过记忆并重新绘制数个图形来增加 Hack 获得奖励道具的可能性的操作

# Hack
入侵，从 Portal 获得道具的操作

# Heat Sink
一种能降低两次 Hack 之间冷却时间的 Mod

# Ingress
浸视界，本游戏的名称

# Install
安装，把 Mod 放置在 Portal 上的操作

# Intel Map
用于查看各地 Portal、Link、Field 状态，并且具有规划 Link 功能的在线地图，网址为 [https://ingress.com/intel](https://ingress.com/intel)

# Inventory
道具仓库

# JARVIS Virus
俗称绿毒，用于把 Resistance 控制的 Portal 转为 Enlightened

# Link Amp
一种能增加 Link 距离的 Mod

# Media
一种指向 Youtube 官方战报视频的道具

# Mind Unit
心元，俗称 MU，Field 的面积单位

# Mod
用于改变 Portal 属性的道具的总称

# Multi-hack
一种能增加 Hack 承载次数的 Mod

# Navigate
导航，在 Scanner 界面显示所要到达位置的距离和方向，并且有语音提示

# OPS
菜单按钮

# Power Cube
俗称 PC、糖，存储大量可用于 Recharge Scanner 的 XM

# Portal
门泉，俗称 Po，游戏中的据点，常常为真实存在的明显地物，所有游戏动作均基于 Portal 进行

# Recharge
充能，俗称充电，使自己的 Scanner 或某个 Portal 上的 Resonator 补充能量的操作

# Resistance
抵抗军，俗称蓝军，阵营之一

# Resonator
谐振器，俗尘脚、电池、Res，用于占领 Portal 并增加其能量

# Scanner
扫描器，装有 Ingress 应用程序的移动设备

# Shield
盾，一种能增强 Portal 防御属性的 Mod

# Turret
一种能增加对敌对阵营 Agent 攻击频率的 Mod

# Ultra Strike
俗称 US，一种作用范围更小，能量更大的异质脉冲武器变体

# Upgrade
使用较高等级 Resonator 替换较低等级 Resonator 的操作

# XM
异质，Exotic Matter 的简称，游戏剧情中的未知能量，从 Portal 进入现实世界，用于支持 Scanner 工作、进行操作和攻击等

# XMP Burster
俗称 XMP、炸弹，一种异质脉冲武器
